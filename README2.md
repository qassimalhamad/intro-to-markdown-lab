## 2 . Anotomy (CSS)
The anatomy of _*CSS*_ style syntax applies to the two methods of incorporating CSS into a web-page:
* **Method 1** is using the attribute of ***style*** inline with the element

***Example***
```
    <p style = "color : blue;"> Hello World</p>
```
![Describion of method 2](https://raw.githubusercontent.com/Codecademy/docs/main/media/css-anatomy-2.png)

* **Method 2** is using a ***seperate file*** of css and ***link*** it to your ***html*** file

***Example***
```
link rel="stylesheet" href="style.css">
```
